🎯 COMO FUNCIONA

┌────────────────┐
│ Seu Celular    │
│ Recebe SMS/    │
│ Notificação    │
└────────┬───────┘
         │ "Debora Ribeiro enviou um Pix de R$20,00 para voce."
         ↓
┌────────────────────────────┐
│ Macrodroid                 │
│ (no celular)               │
│ Capta automaticamente      │
└────────┬───────────────────┘
         │
         ↓ POST /api/webhooks/notifications
┌────────────────────────────┐
│ Webhook (seu servidor)     │
│ Recebe notificação         │
│ Valida família             │
└────────┬───────────────────┘
         │
         ↓ Chama parseNotificationWithAI()
┌────────────────────────────┐
│ OpenAI (Claude)             │
│ "Interprete esta mensagem" │
│ Retorna JSON estruturado   │
└────────┬───────────────────┘
         │
         ↓ {amount: 20, type: INCOME, account: PicPay, ...}
┌────────────────────────────┐
│ Banco de Dados             │
│ Cria Transaction           │
│ Status: PENDING            │
└────────┬───────────────────┘
         │
         ↓
┌────────────────────────────┐
│ Dashboard                  │
│ Mostra "Pendentes"         │
│ Você revisa                │
└────────┬───────────────────┘
         │
         ├─→ ✓ Confirmar → Status: CONFIRMED ✅
         │
         └─→ ✗ Rejeitar → Deletada

📞 REFERÊNCIA RÁPIDA

Webhook:
  POST /api/webhooks/notifications

Filtrar PENDING:
  GET /api/transactions?status=PENDING&source=NOTIFICATION

Confirmar:
  PATCH /api/transactions/:id { status: "CONFIRMED" }

Rejeitar:
  DELETE /api/transactions/:id
